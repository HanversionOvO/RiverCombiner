## Description

首页与导航模块是 river 应用的骨架层，基于 Material Design 构建应用外壳和底部标签导航体系。`RiverApp` 作为应用根 Widget，在 `main.dart` 中完成初始化，配置主题、路由和全局依赖注入。

`HomeShellPage` 是底部标签栏的容器页面，承载帖子列表、搜索、通知、个人中心等主要 tab 的切换。每个 tab 通过 `TabPlaceholderPage` 作为懒加载占位，仅在首次切换时初始化对应模块，优化启动性能。`RiverPageRoute` 提供自定义页面路由转场动画，区别于默认的 Material 路由过渡效果，同时 `RiverRouteObserver` 追踪页面导航事件用于埋点和调试。

依赖注入由 `AppDependencies` 统一管理，作为 Provider 向 Widget 树注入全局服务实例（网络客户端、存储服务、配置等）。`PostsStartupPreloadStore` 在启动阶段预加载帖子数据缓存，结合应用启动动画序列，通过 tunneling 机制在启动画面和主页之间实现无缝过渡，确保用户进入应用后即刻看到内容。
