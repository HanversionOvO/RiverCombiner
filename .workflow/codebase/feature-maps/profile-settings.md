## Description

The User Profile & Settings feature encompasses both the outward-facing profile experience and the comprehensive settings system of the river platform. `MinePage` serves as the user's personal hub, displaying their activity feed, achievement badges, and quick-access shortcuts to settings and profile editing. `RiverSideProfilePage` presents the platform-level profile view -- the public-facing page that other users see when visiting a member's profile, including their post history, join date, and community contributions.

The settings architecture is organized around `AppSettingsController`, a centralized controller that manages persistent user preferences across multiple domains. Through `SettingsPages`, users can configure their AI provider preferences (selecting between available LLM backends and adjusting generation parameters), customize appearance (font family, font size, corner radius, and theme), manage server connection configuration, control cache storage limits and eviction policies, configure image hosting service preferences, and toggle push notification channels. `RiverSideAccountSettingsModels` and `RiverSideProfileModels` provide the typed data layer backing both settings persistence and profile data, including serialization for remote sync.

Ancillary components round out the feature: `MineSettingsAppBar` delivers a consistent settings navigation bar with contextual actions, while `MineQRPages` generates and displays QR codes for profile sharing, enabling quick user discovery via in-person scanning. The app icon customization system allows users to select from a curated set of alternative app icons, with changes applied through the platform bridge.
