## Description

话题详情模块为 river 提供完整的帖子详情浏览体验，涵盖正文渲染、评论区交互、表情反应、投票和 AI 摘要等核心功能。`TopicDetailPage` 作为主页面，拆分出独立的 actions、loading、reactions 子模块处理交互逻辑、加载状态和表情反应面板。

内容渲染由 `TopicDetailWidgets` 组件族承担，按职责分为 cards（帖子卡片容器）、content（Markdown 正文渲染）、images（图片展示）和 meta（元信息如作者、时间、板块）四个维度。`TopicDetailContentUtils` 提供文本处理、格式转换等工具函数，辅助内容层的排版与展示。

评论系统通过 `CommentDetailPage` 实现楼中楼线程化评论，支持评论的嵌套查看、回复和交互操作。`RiverSideApiClientReactions` 封装表情反应的 API 调用，包括反应列表获取、添加/取消反应等，配套 `RiverSideApiClientParsingReactions` 负责响应解析。

`RiverImageViewer` 提供内联图片的点击放大查看能力，支持缩放手势和图片画廊浏览，同时服务于 FT-010 基础设施层。`RiverSideTopicModels`（TC-015，共享自 FT-003）中的数据模型如 `RiverSideTopicDetail`、`RiverSideTopicPostDetail`、`RiverSideTopicPoll` 和 `RiverSideAiTopicSummary` 支撑投票结果解析和 AI 生成摘要的展示。整体架构以 Page 为中心，将复杂的内容页面拆分为可组合的 Widget 和 Utils 模块，保持各职责边界清晰。
