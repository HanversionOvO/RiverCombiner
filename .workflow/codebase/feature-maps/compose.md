## Description

发帖编辑器模块为 river 提供富文本 Markdown 编辑能力，以 `ComposeTopicPage` 为核心页面，拆分 actions 和 view 子模块实现编辑操作和 UI 渲染的解耦。`RiverMarkdownEditor` 是编辑器的核心 Widget，支持 Markdown 语法输入、在线预览、@提及用户、话题标签和草稿管理（`RiverMarkdownDraftEntry`）。

AI 辅助写作由 `RiverAiService` 驱动，提供多个 AI 场景（`RiverMarkdownAiScene`）如润色、扩写、摘要和翻译。用户可通过编辑器内置的 AI 操作入口提交写作请求（`RiverMarkdownAiRequest`），AI 返回的结果可直接插入或替换当前编辑内容。`RiverAiService` 同时服务于 FT-010 基础设施层。

图片上传由 `PicUiImageHostService` 集成 PicUI 图床，支持上传选项配置、多图链接管理、相册浏览等功能。用户可通过编辑器插入图床中的图片，或在编辑过程中直接拍照/选择相册上传。

发布前提供 `ComposeTopicPreviewPage` 进行内容预览，展示 Markdown 渲染后的实际效果。板块/分类选择通过两级 Picker 实现：`RiverPublishCategoryPickerSheet` 处理发布时的分类选择，`RiverSideCategoryPickerSheet` 配合 `RiverSideCategoryLoader` 完成服务端分类数据的异步加载，底层由 `Categories`（`RiverSideCategoryStore`）管理分类数据的缓存和分组展示。编辑器内置草稿自动保存机制，确保用户在意外退出时不会丢失编辑内容。
